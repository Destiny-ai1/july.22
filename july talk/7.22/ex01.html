네트워크의 시작 = LAN (Local Area Network)

  1. 저장장치 공유
  2. 프린터 공유

인터넷등장

  1. LAN들을 연결
      2. 중간구간이 끊키더라도 우회해서 연결 가능

  2. router: 데이터의 목적지가 우리면 흡수, 아니면 목적지 쪽으로 pass

인터넷과 서비스

인터넷을 위한 통신 규약 : TCP/ IP

  • 이메일 서비스 : smtp 프로토콜
  • www 서비스 : http 프로토콜
  • 파일 다운로드 : ftp 프로토콜

    • internet , intranet , extranet

    • internet : TCP/ IP 를 사용하는 LAN
    • extranet : 인트라넷 을 암호화 된 장비로 연결
